# Plinko

### Stack
- JS
- PixiJS
- GSAP
- Chroma-js
- MersenneTwister
- Webpack

### Run
`npm install`

`npm run start`

 Go to: http://localhost:3300/


### Demo
https://ivan-kitsa-dev.github.io/Plinko
